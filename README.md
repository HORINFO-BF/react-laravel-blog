## Installation

### Composer
```bash
composer install
```
### Yarn ou npm
```bash
yarn
#ou
npm install
```

## Après installation

### Renommer le fichier ".env.exemple" en ".env" et générer une clé avec la commande:

```bash
php artisan key:generate
```

### Migrer les tables du projet:

```bash
php artisan migrate
```
